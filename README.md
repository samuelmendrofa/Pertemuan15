## 🚀 Features

* ✅ **Add Task**
  Tambahkan tugas baru dengan mudah.

* ✏️ **Edit Task**
  Ubah isi task yang sudah ada.

* 🗑️ **Delete Task**
  Hapus task dengan konfirmasi alert.

* 🔍 **Search Task**
  Cari task berdasarkan kata kunci.

* 🎯 **Filter Task (All / Active / Done)**
  Mempermudah pengelompokan task.

* 🌙 **Dark Mode / Light Mode**
  Tampilan bisa disesuaikan dengan preferensi pengguna.

* 💾 **Persistent Storage (AsyncStorage)**
  Data tetap tersimpan walaupun aplikasi ditutup.

* 📊 **Progress Tracking**
  Menampilkan persentase task yang telah selesai.

## ⚙️ Installation & Setup

```bash
# Clone repository
git clone https://github.com/samuelmendrofa/MyTaskList-App.git

# Masuk ke folder project
cd Mytasklist-App

# Install dependencies
npm install

# Jalankan aplikasi
npx expo start
```
